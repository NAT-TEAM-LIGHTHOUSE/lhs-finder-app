# Privacy Policy – LHS App Finder

LHS App Finder is a Chrome extension that allows users to search and open Lighthouse (LHS) internal applications directly from the Chrome address bar.

## Data Collection and Usage
LHS App Finder does not collect, store, or sell any personally identifiable information.
The extension sends user-entered search queries to a configured backend service only for the purpose of retrieving application search results.
No browsing history, webpage content, or sensitive user data is accessed or tracked.

## Data Storage
The extension may store basic configuration values and user preferences locally using Chrome’s storage API. This data remains on the user’s device and is not shared externally.

## Data Sharing
LHS App Finder does not sell, rent, or transfer user data to third parties outside of the approved use cases required for the extension’s core functionality.

## Security
All network requests are initiated by explicit user action. No background tracking or automated data collection occurs.

## Changes to This Policy
This privacy policy may be updated if the extension’s functionality or legal requirements change.

## Contact
For questions or concerns regarding this privacy policy, please contact the extension developer.
